# Nota Rapida Desktop

A minimal chromium wrapper over the 'Nota Rapida' web application to provide a desktop feel.

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
